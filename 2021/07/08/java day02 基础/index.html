01 变量、作用域、常量

1.1 变量

  1. 变量的格式： 数据类型 变量名 = 值
  2. 注意事项：
    • 数据类型可以是基本类型，还可以是引用类型
    • 变量名必须是合法的标识符
    • 变量声明必须以;结尾

1.2 变量作用域

1.3 常量

  1. 常量初始化后，值不能再改变
  2. 初始化格式：final 常量名 = 值；
  3. 常量名都用大写字母

命名规则

  • 类成员变量：首字母小写+驼峰原则：除了第一个单词外，后面单词首字母大写。xiaoFang
  • 局部变量：首字母小写+驼峰原则
  • 常量：大写字母+下划线：XIAO_FANG
  • 类名：首字母大写+驼峰原则
  • 方法名：小写+驼峰原则
