概念的总结

swarm

集群的管理和编号，docker可以初始化一个swarm集群，其他结点可以加入。（管理，工作者）

Node

就是一个docker结点，多个结点就组成了一个网络集群（管理、工作者）

Service

任务，可以在管理结点或者工作结点来运行。核心，用户访问。

Task

容器内的命令、细节任务！

逻辑是不变的。

命令 -> 管理 -> api -> 调度 -> 工作节点（创建Task容器维护创建！）
