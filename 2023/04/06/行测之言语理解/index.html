行测之言语理解
Fang Lv7
  2023-04-06 10:33:07 2023-04-06 10:33

 4k Words  13 Mins  

一、片段阅读

1、中心理解题

  1. 题型特征：题干中出现“主要/旨在/重在/意在/想要说明（论述、强调）/主旨/主题/观点/概括”等标志词。
  2. 解题思路：寻找中心句或整体把握文段中心
    • 看提问方式，辨别题型
    • 边读边勾画，明确中心句式话题
    • 中心句或话题，匹配选项，对比择优
  3. 对比择优：优先匹配话题，再去对比细节（差中选优、优中选优）
  4. 干扰选项特征：
    • 非文段重点，非文段核心话题
    • 无中生有

1、转折关系

  1. 答题要点：
    • 转折之后为重点
    • 巧用“引用+转折”验证答案，加强理解
  2. 典型关联词：虽然····但是···、尽管·····可是····、不过、然而、却、其实/事实上/实际上
  3. 转折文段中，除了单纯的考察一个转折关联词，还会出现多个关联词情况，所以要根据实际情况，结合行文脉络判定文段的重点
  4. 错误选项特征：
    • 转折前的内容
    • 围绕例子的表述
    • 围绕原因的表述
    • 无中生有

2、因果关系

  1. 答题要点：
    • 结论是重点
    • 注重选项设置：·····的原因，匹配结论话题即可
    • 宏观的代词（一个词可以指代上文所有内容）可以引导结论
  2. 标志词：
    • 最常考：因此、因而、故而、所以、从而
    • 易忽略：可见、看来
    • 文段选项均可能出现：导致、致使、使得、造成
  3. 主体、话题法：
    • 主体：文段着重论述的，所围绕论述的对象（人、事物、概念）
    • 话题：与主体相关的事件
  4. 结论之后有其他关联词：当结论之后出现转折等关联词时，需要结合整个文段的行文脉络进行把握

3、必要条件关系

  1. 标志词：只有····才·····
  2. 理论要点：必要条件时重点，即只有和才之间的内容为重点
  3. 同义替换：只有A才B=A是B的必要条件=不A则不B=A是B的前提、基础、保障、方法
  4. 解决问题的对策是重点
  5. 对策的标志词：
    • 应该、应当、应、需要、要
    • 通过/采取···手段/途经/措施/方法/方式/渠道，才能····
    • 前提、基础、保障
    • 负有···的义务/····必由之路/势在必行/要领在于·····
  6. 如果遇到“提出问题+分析问题“结构的文段，那么要优先能解决问题的对策
  7. 反面论证：
    • 典型格式;如果、一旦、倘若····+不好的结果，除非····否则
    • 使用方法：将【如果】、【倘若】之后的内容反过来，即为正确的做法，有【否则】的句子，直接看【否则】之前
  8. 行文结构给对策：
    • 提出问题-分析问题-解决问题
    • 提出问题-解决问题
    • 问题+精准原因+严重后果（对策要基于话题）
  9. 如果文段有明确的对策，直接抓住对策即可快速把握文段中心
  10. 文段只出现问题时，解决问题可能出现在选项中作为正确答案

4、并列关系

  1. 答题要点：不分主次、不可偏颇、全面概括（并列加和，归纳共性）
  2. 文段标志：出现“此外、同时、另外、；”等标志词、标点，或文段中无明显关联词，或句式相近。
  3. 隐形并列：并列层次（判断标准：谁跟谁一伙）
  4. 错误选项特征：表述片面
  5. 除了典型的标志词外，针对同一主体在不同领域或者不同方面的论述，也属于并列结构
  6. 如果文段的中心句为并列结构，需概括全面完整
  7. 并列文段通常包括多方面的情况，要全面概括，选项中会出现“许多、一系列、各种”等标志词，一定要注意。
  8. 除了典型的标志词外，时间顺序也可以引导并列结构，而解题的关键依旧是全面概括
  9. 文段中的总与分句子关系决定文段结构：
    • 总：观点、结论、对策
    • 分：铺垫背景、举例子、解释原因、解释概念、解释过程、找证据（实验、报告、调查、研究、数据）
  10. 如果文段中有两个主题词，正确答案必须同时包括这两个主题词

5、程度词

  1. 程度词所在的语句通常为文段重点，特别是程度词引导的名词往往是文段主题词
  2. 标志词：
    • 典型程度词：更、最、尤其、根本、正是、真正等
    • 非典型程度词：致命、无疑是、罪魁祸首、首要方式

6、行文脉络

  1. 中心句特征：形式（重点词特征）、内容（观点，一般情况下是明确的）
  2. 脉络相对简单的，是典型“提出观点+举例论证”的题目，首句观点即为文段中心
  3. 分总结构：代词（这、此）出现在末尾，起到总结前文的作用
  4. 分述句的作用是对文段重点内容进行解释说明或加强论证（举例子、数据资料、正反论证、原因解释、并列分述）
  5. 当文段中出现由“比如、例如、譬如”等词引导的例子时，可快速阅读，且围绕例子设置的选项不是文段重点，应当排除

2、标题填入题

  1. 标题填入题：给文段选择恰当的标题（体现文段重点）
  2. 逆向思维：警惕范围过大的选项
  3. 文段中出现并列结构时，出题人常常将其中的一部分内容，即片面的表达，设置为干扰选项。
  4. 标题的选择应首先匹配文段的中心
  5. 故事类的文段通常会采用“开端-发展-高潮-结局”的结构，一般而言，高潮和结局部分比较重要

3、细节判断题

  1. 题型特征：题干中出现“理解正确/不正确”“符合/不符合”等标志词
  2. 警惕细节判断题选项设置的四大陷阱：
    • 无中生有：文中没有，选项偏说有（比较关系、因果关系）
    • 偷换概念：替换、混搭、数量、时态、概念范围扩大、语气
    • 偷换逻辑：强加因果、因果倒置
    • 偷换时态：过去（已经、了、完成）、现在（正在、在···中、着）和将来（将要、立刻、以后、趋势）
  3. 选项中出现绝对化表述要警惕
  4. 相对绝对法：迅速搞定细节题的重要法宝
    • 绝对标志词：一定、全部、完全、所有、均、永远、任何、取代、主导、唯一、只要····就····、无论···都···等
    • 相对标志词：可能、也许、大概、一般、尽量
    • 如需选择“正确/相符的”，包括相对标志词的选项作为正确答案，或可优先验证
    • 如需选择“错误/不相符的”，包括绝对标志词的选项作为正确答案，或可优先验证
  5. 细节查找：
    • 查找原因（要素）类：根据文意，影响····的因素是/····的关键是····，
    • 定位查找对象，就近查找，直接确定
    • 查找目的/意义：文段中提到···是为了证明
    • 查找例子，引用所论证的观点/结论
  6. 解题思路：根据选项长短
    • 选项短：先浏览选项，后阅读文段，排除+确定
    • 选项长：扫一眼选项（选项有明确的定位功能-摇摆式、定位、排除+确定；选项中无明确的定位功能-读完文段后再作答（划分话题群））

4、词句理解题

  1. 题型特征：这段文字中“某某”指的/含义/意思是·····
  2. 若词句出现在文段首位，则需整体把握文段内容
  3. 如果词句前出现“：”“——”等表示解释说明的标点符号，则重要理解其前面的内容。
  4. 若所需理解的词语为代词，则主要理解代词之前的内容

二、语句表达

1、语句排序题

  1. 考察目的：语句排序题即按正确顺序重新排列文段给出的语句
  2. 充分利用选项提示，排除法解题
  3. 认知逻辑：由整体到细节
    • 介绍概念，整体描述
    • 详说过程/细节（事情发展的顺序）
    • 前提（语句基于同一话题）

1、首句特征

  1. 常见首句特征：
    • 背景引入（随着、近年来、在····大背景/环境下）
    • 下定义（····就是/是指）
    • 提出观点（有人说、人们普遍认为）
  2. 非首句特征：
    • 指代词单独出现（结论类、补充类表述）
    • 关联词后半部分

2、确定捆绑集团

  1. 关联词使用方法：
    • 配套出现：根据配套出现的关联词可直接将两句话捆绑
    • 单独出现：根据关联词的作用寻找另一半，转折找相反句式，并列则找相同句式
    • 话题捆绑
    • 逻辑关系捆绑：因果关系、转折关系、递进关系
  2. 指代词：
    • 标志词：它、他、他们、这、那
    • 使用方法：找准指代词指代的内容，将两句话进行捆绑
    • 当时，也是一种代词，可以用来指代时间

3、确定顺序

  1. 时间顺序：语句中出现多个时间，按照时间发展脉络排列先后顺序
  2. 逻辑顺序：
    • 前文出现A和B，后文基本会按照先A后B的顺序展开讨论
    • A和B即可能出现在首句，同样也可能出现在尾句
    • 不同概念之间会有内在联系，如从宏观到微观、从整体到局部、从大到小、从前提到结果，层层深入，层层递进

4、确定尾句

  1. 引用结论和提出对策的语句通常适合作为尾句
  2. 典型标志词：因此、所以、看来、这+应该、需要
  3. “这些”除了可以作为指代词寻找绑定集团外，在文段结尾可以起到引导结论、总结全文的作用。

2、语句填空题

  1. 题型特征：填入画横线部分最恰当的一项是····（所填入的句子是对上下文原有信息的重复）
  2. 解题思路：捋顺文段话题，根据上下文限定作答
  3. 最忌讳：想当然、无中生有、新信息
  4. 做题技巧：
    • 对比择优–和上下文衔接紧密程度、具体话题、倾向等
    • 没有心仪选项–考虑同义替换、表达、倾向等，排除法择优
    • 选项长，文段短–抓大放小，精选话题
    • 内容特殊、选项多为诗歌、名言、谚语–提炼主题、话题
    • 形式特殊–换汤不换药，从容排除，确定答案

1、横线在首尾

  1. 横线位置：文段结尾，且为完整语句
  2. 句子作用：往往都是中心句
  3. 解题原则：需要对整个文段的内容进行全面概括和总结
  4. 横线在结尾（总结前文、提出对策；保证前后主语或话题一致）

2、横线在中间

  1. 横线位置：位于文段中间
  2. 句子作用：衔接上下文、注意引出下文新话题
  3. 横线位于文段中间时，用代入排除法解题速度更快
  4. 除了要注意上下文衔接，还要注意横线前后出现的关联词

3、接语选择题

  1. 接语选择题，就是给出一段文字，要求考生在此基础上推测出下文，注意审题、注意表达倾向、根据新话题预测下文
  2. 文段结构为“提出问题+分析问题”，而解决问题往往出现在选项中
  3. 尾句为解决问题的对策时，后文往往会围绕对策部分展开具体论述
  4. 根据行文脉络，预测下文：
    • 上文有问题，下文要解决
    • 上文提出观点+解释说明，下文围绕观点继续说；如果没有围绕观点表达的，则排除无关项旧话题

三、逻辑填空

  1. 考察实质：所填词语和语境信息的匹配程度（对比择优/组团择优）
  2. 词语加双引号的情况：
    • 引用其字面指代含义，扩大搭配范围，使表达更形象
    • 反讽（不常见）

1、词的辨析

1、词义侧重

  1. 角度一：语素差异、构词拆分
    • 长相相近拆语素，拆分语素，拆分构词—明确词语差异、侧重点
    • 理解语境，明确需求
    • 拆分构词，对比择优
  2. 角度二：程度轻重
    • 意思相近看程度：依托选项提示，辨别考点
    • 依据选项提示，明确考点
    • 根据语境的轻重程度，对比择优
  3. 面对长得特别像、发音也相似的词语，考生要重点区分两个词语中的不同部分，辨别方法有：
    • 拆字组词：用不一样的字单独组词（推脱：脱可以组词摆脱）
    • 整词搭配：用整个词进行组词搭配（推脱责任）

2、固定搭配

  1. 根据词性找搭配：
    • 若选项都是动词，回到文段找名词或副词
    • 若选项都是名词，回到文段找动词或形容词
  2. 修饰成分：
    • __ 的 __
    • __ 地 __
    • 保持一致形成搭配
  3. 一个词语同时搭配多个词语
    • 提示标志：多由并列关联词引导
    • 使用方法：所填词语要与多个词语均能搭配
  4. 找准搭配对象

3、程度轻重

  1. 词义程度轻重是指意义相近的词语在辨析程度和性质上有轻重的差别
  2. 象征一般指用具体事务表现抽象含义：玫瑰象征爱情、鸽子象征和平

4、感情色彩

  1. 感情色彩是指词语所反映出来的人对某事某物的情感倾向、态度、评价等，分为褒义（成果）、中性（结果）、贬义（后果）三种类型
  2. 所填词语的感情色彩与文段的感情色彩保持一致

2、语境分析

1、关联关系

  1. 转折关系：__ 但/然而/却 __
    • 理论要点：转折关联词前后语义相反、感情色彩相反
    • 标志词：但是、可是、然而、却
  2. 因果关系：__ 所以/因而 __
    • 理论要点：横线前后构成因果关系
    • 标志词：因为····所以····、由于·····因此等
  3. 并列关系：
    • 标志词：同义并列（“、”“，”），反义并列（”不是····而是···“”是·····不是····“”相反“”反之“）
    • 理论要点：同义并列（语义相近），反义并列（前后语义相反或相对）
    • 句式相同或相近，要结合前后语境来分析
    • 特别注意：三个句子，句式一致：服从整体语境的前提下，根据其他两个考虑是否形成差异化
    • 并、并且（优先充当递进关系）

2、对应关系

  1. 解释类对应：
    • 题干特点：分句，______，分句，即文段中有分句对横线处所填词语进行解释说明
    • 标志词：即、就是、无异于、比如、例如、冒号（:）、破折号（——）等。
    • 使用方法：准确找到与横线处所填词语想对应的句子，理解句意，对应选项
  2. 要对“无异于”“无疑是”等解释类对应的标志词保持敏感，当这一类标志词出现时，大家可以从前文中找到对应内容
  3. 限定照应：
    • 背景—->结论（因果）
    • 讨论前提—–>讨论内容（两头都要顾）
    • 条件—–>结果（只有·····才能······）
  4. 形象性照样
  5. 重点词句对应：
    • 形象化表达，如拟人、类比等形象表达
    • 指代词
    • 主题词，即文段主要围绕论述的核心名词

三、篇章阅读

  1. 答题策略：
    • 标注段落号
    • 看问题，边读边做
    • 顺序后置的题型：针对全文的(中心理解题，细节题)
    • 如果第一题时针对全文的细节理解，则可先看选项
    • 观看没有用到的段落，也许会出现所需的信息点
